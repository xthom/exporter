exporter
========

Note: currently only for MySQL

1) installation:
  npm install

2) set-up the config (add as many dbs you want)

3) display configured db list
  ./bin/exporter list
  
4) export database
  ./bin/exporter db_name [--dir=/my/dir/path]

5) or export all databases in config file
  ./bin/exporter
